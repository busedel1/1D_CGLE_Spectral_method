◆Contents
waveeq_cpu_gpu.zipの中には以下のファイルが含まれております;
-	plotWave.m    : 計算過程を3D描画するコード
-	readme.txt      : readmeファイル
-	WaveEqn_CPU.m : WaveScript.m/ 
WaveEqnGUI.mに呼ばれるCPU上で波動方程式を解く関数
-	WaveEqn_GPU.m : WaveScript.m/ 
WaveEqnGUI.mに呼ばれるGPU上で波動方程式を解く関数
-	WaveEqnGUI.fig : GUIのレイアウトを収めたFIGファイル
-	WaveEqnGUI.m : GUI上で波動方程式を解くためのメイン関数
-	WaveScript.m : スクリプトとして波動方程式を解くためのメイン・プログラム
